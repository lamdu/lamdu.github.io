Lamdu is an open-source programming language which aims to be both useful and delightful

Lamdu aims to achieve its goal by:

  • Providing a text-like user interface with rich code completions, without the possiblity of syntax errors
  • Automating code layout with responsive design, without the user needing to deal with manual formatting
  • Harnessing the power of static types, while making them friendly by employing type inference and a novel user interface for type errors
  • Enabling safe live reloading by differentianting pure computations from effectful processes
  • Making the natural language of the code just a display preference

Status

The current release is a technology preview and isn't yet recommended for production use. Creating a programming language that isn't based on text-files is no small feat. The intention is for it to become a competetive, fully-featured programming language and environment but it will still take us a while.
